# Implementation Plan — Secure OTP-Gated File Upload Portal

> Generated by bicep agent | 2026-04-06

## Summary

This plan describes the Bicep infrastructure implementation for the secure OTP-gated file upload portal. All modules use Azure Verified Modules (AVM) where available. The Logic App workflow definition uses raw Bicep (no AVM module exists for Logic App Consumption workflows).

---

## Module Structure

```
scenario/secure-upload-portal/infra/
├── main.bicep                      # Orchestrator — deploys all modules
├── main.bicepparam                 # Parameter file for azd
├── abbreviations.json              # Resource naming abbreviations
└── modules/
    ├── network.bicep               # VNet, subnets, NSGs
    ├── storage.bicep               # Storage account, private endpoint, DNS zone
    ├── logicapp.bicep              # Logic App Consumption + HTTP trigger workflow
    ├── monitoring.bicep            # Log Analytics + Application Insights
    └── appservice.bicep            # App Service Plan + App Service + role assignments
```

## Resource Deployment Order

```
1. monitoring.bicep       — Log Analytics Workspace, Application Insights
2. network.bicep          — VNet, subnets, NSG
3. storage.bicep          — Storage Account, Private Endpoint, DNS Zone (depends on network)
4. logicapp.bicep         — Logic App Consumption workflow
5. appservice.bicep       — App Service Plan, App Service, MI role assignments, app settings
```

## Key Design Decisions

| Decision                            | Implementation                                                     |
| ----------------------------------- | ------------------------------------------------------------------ |
| AVM for App Service Plan            | `br/public:avm/res/web/serverfarm:0.4.0`                          |
| AVM for App Service                 | `br/public:avm/res/web/site:0.12.0`                               |
| AVM for Storage Account             | `br/public:avm/res/storage/storage-account:0.14.0`                |
| AVM for VNet                        | `br/public:avm/res/network/virtual-network:0.5.0`                 |
| AVM for Log Analytics               | `br/public:avm/res/operational-insights/workspace:0.9.0`          |
| AVM for App Insights                | `br/public:avm/res/insights/component:0.4.0`                      |
| Logic App / Private Endpoint / DNS  | Raw Bicep (no suitable AVM module)                                 |
| Storage access                      | System-Assigned MI + `Storage Blob Data Contributor` (RBAC)       |
| Storage network                     | Private Endpoint (blob), Public access disabled                    |
| OTP delivery                        | Logic App HTTP trigger → O365 Outlook connector                   |
